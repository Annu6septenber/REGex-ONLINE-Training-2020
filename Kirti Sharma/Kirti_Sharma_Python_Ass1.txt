Ques1: What is JPython & CPython?
Ans: ~ Jython is the JVM implementation of the Python programming language. It is designed to run on the Java platform
     ~ CPython is the reference implementation of Python written in the C programming language.
      

Ques2: Basic difference between Python2 & python3
Ans: 1- In python2, the function which takes the data from user is user_input() while in python3, the function is input().
     2- python2 supports ASCII format while python3 supports Unicode.  


Ques3: Difference between ASCII & unicode.
Ans: ASCII and Unicode both are standards on how to represent different characters in binary. The basic differnce between them is :
     ASCII defines 128 characters, which map to the numbers 0–127 while Unicode defines (less than) 221 characters, which, similarly, map to numbers 0–221.
